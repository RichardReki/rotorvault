⑥ 常见问题

现象原因 / 解决
找不到路径 F:\f\Hacks\...你在 PowerShell 里。/f/... 只有 Git Bash 认。改用 Git Bash。
无法识别 forgePowerShell 里没有 forge 的 PATH。用 Git Bash（标题 MINGW64）。
否决事件标成 “Decode input data”Blockscout 的标签，内容其实是事件解码，正常。看 “Call: Rebalanced(...)” 即可。
forge test 等 ~20 秒fork 测试在连真实 Coston2，正常。后期把等待段跳剪到 3–5 秒。
Blockscout 加载慢先刷新等页面出来再开录；兜底可展示 README 里的证明表格。
reproduce.sh 报错兜底：python scripts/verify.py（输出同样的 PASS 行）。
